GUIs
====

PIEC provides Graphical User Interfaces (GUIs) for running common experiments interactively,
without writing any code. They are a good starting point before moving to notebooks or
scripted workflows.

Launching a GUI
---------------

GUIs live in the ``piec/guis/`` directory of the repository. With PIEC installed, launch one
from your terminal:

.. code-block:: console

   python path/to/piec/guis/gui_name.py

.. note::
   Check the ``piec/guis/`` directory for all available GUI files. Each file corresponds to a
   different experiment type.
